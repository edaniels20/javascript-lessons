Calculating PTO

Version 1

  1. start
  2. set weeksWorked = 52 / 2
  3. totalPto = weeksWorked * 5
  4. Display "Your total Pto for this year is " totalPto
  5. end

Version 2

  1. Start
  2. Display "How many years have you worked?"
  3. Input yearsWorked
  4. if yearsWorked < 5 Set hours = 5
  5. if yearsWorked > = 5 and < = 10 set hours = 6
  6. if yearsWorked > 10 set hours = 7
  7. Display "Your hours of Pto are " (52/2) * hours
  8. Finish
